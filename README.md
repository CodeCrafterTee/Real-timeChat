# Real-time Chat Application 

A modern, **real-time chat web application** built with ASP.NET Core MVC, SignalR, EF Core, and SQL Server.

---

## Features

**User registration & login**  
 **Real-time messaging** with multiple chat rooms  
 **Direct/private messaging support**  
 **Typing indicator** shows who is typing  
 **Dark/Light mode toggle** for UI  
 **Message history persisted in SQL Server**  
 **Optional emoji support**  
 **Responsive and modern design** using CSS  

---

## Technologies

- **Backend:** ASP.NET Core MVC, SignalR  
- **Database:** EF Core + SQL Server  
- **Authentication:** ASP.NET Core Identity  
- **Frontend:** HTML/CSS + optional Bootstrap  
- **Real-time features:** SignalR hubs, group messaging
